# 🤝 Como Contribuir

Obrigado por considerar contribuir para o projeto do Espaço Bem Estar!

## 📋 Processo de Contribuição

### 1. Reportar Bugs

Se encontrou um bug:

1. Verifique se já não existe uma issue sobre isso
2. Crie uma nova issue com:
   - Título descritivo
   - Descrição detalhada do problema
   - Passos para reproduzir
   - Screenshots (se aplicável)
   - Navegador/dispositivo utilizado

### 2. Sugerir Melhorias

Para sugerir melhorias:

1. Abra uma issue com a label `enhancement`
2. Descreva a melhoria e seu benefício
3. Se possível, inclua mockups ou referências

### 3. Enviar Código

```bash
# 1. Faça um fork do repositório

# 2. Clone seu fork
git clone https://github.com/SEU-USUARIO/espaco-bem-estar.git

# 3. Crie uma branch
git checkout -b feature/nome-da-feature

# 4. Faça as alterações e commit
git add .
git commit -m "✨ Add: Descrição da feature"

# 5. Envie para seu fork
git push origin feature/nome-da-feature

# 6. Abra um Pull Request
```

## 📝 Convenções de Código

### Commits

Use [Conventional Commits](https://www.conventionalcommits.org/):

- `✨ feat:` - Nova funcionalidade
- `🐛 fix:` - Correção de bug
- `📚 docs:` - Documentação
- `💄 style:` - Formatação/estilo
- `♻️ refactor:` - Refatoração
- `⚡ perf:` - Performance
- `🧪 test:` - Testes

### CSS

- Use as variáveis do design system
- Nomeie classes em inglês
- Siga o BEM para componentes complexos

### HTML

- Use tags semânticas
- Mantenha acessibilidade (alt, aria-labels)
- Indente com 2 espaços

## 🎯 Prioridades

1. Acessibilidade (WCAG 2.1 AA)
2. Performance (Core Web Vitals)
3. SEO
4. Responsividade
